1Weight card — 30d sparkline + 7d MA delta

Sparkline shows 30 days of daily readings (faint thin line) overlaid with a thicker 7-day moving-average curve in mint. The mint curve is what's "real" — daily lines are noise.

big number = today's logged weight (e.g. 184.2 lb) delta = current 7d MA − 7d MA from 7 days ago sparkline = 30 days raw + 30 days MA chevron = open weight detail screen

In the mockup: today = 184.2 lb. Last 7-day average = 184.4 lb. The 7-day average from 7 days ago = 184.8 lb. Delta = ↓ 0.4 lb — true week-over-week change with noise filtered out.

2Volume card — 4-week mesocycle bars

Sparkline now uses 4 weekly bars (not a smooth line) — each bar is one week's total tonnage, with a connecting line on top showing the trend. More honest visualization for sparse weekly data than a faked line chart.

big number = (last 4wk volume − prior 4wk volume) / prior 4wk × 100 delta = "vs prior 4 wk" (the comparison is the metric) sparkline = each of the last 4 weeks, week-by-week chevron = open volume detail screen

3Stats strip — same-point-in-week (Option B)

Tuesday morning? Sessions compares Mon-Tue this week vs Mon-Tue last week. Once Sunday rolls over, becomes full-week vs full-week automatically. No more partial-week gaslighting.

sessions = count(workouts, this Mon → now) vs (last Mon → equivalent point) PRs = same comparison window tonnage = sum(weight × reps, same windows) arrow = neutral slate, never green/red
