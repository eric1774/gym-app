```typescript
interface TabBarProps {
  tabs: string[];           // e.g. ['Macros', 'Hydration']
  activeIndex: number;      // 0-based index of active tab
  onTabPress: (index: number) => void;
}
```

**Export:** `export function TabBar({ tabs, activeIndex, onTabPress }: TabBarProps)`

**Visual implementation per UI-SPEC and D-01:**
- Container: `flexDirection: 'row'`, full width, `borderBottomWidth: 1`, `borderBottomColor: colors.border` (1px separator)
- Each tab item: `flex: 1`, `alignItems: 'center'`, `justifyContent: 'center'`, `minHeight: 44` (touch target per UI-SPEC)
- Tab is a `TouchableOpacity` with `activeOpacity={0.7}`
- Active tab label: `colors.primary` (#FFFFFF), `fontSize.sm` (13), `weightBold` ('700'), uppercase via `text.toUpperCase()`
- Inactive tab label: `colors.secondary` (#8E9298), `fontSize.sm` (13), `weightBold` ('700'), uppercase
- Active underline: positioned at bottom of tab item, `height: 2`, `backgroundColor: colors.accent` (#8DC28A), full width of tab item. Implement with `borderBottomWidth: 2` and `borderBottomColor: colors.accent` on the active tab item (or a separate View). The cleanest approach: a View inside each tab item at the bottom, height 2, conditionally colored `colors.accent` for active or `'transparent'` for inactive.
- No animation on switch (D-04) — just re-render with new activeIndex
- No heavy background fill or border on individual tabs — underline only (D-01)

**Accessibility per UI-SPEC:**
- Each tab item: `accessibilityRole="tab"`, `accessibilityState={{ selected: isActive }}`

**Styles:** Use StyleSheet.create. Import `colors` from `../theme/colors`, `spacing` from `../theme/spacing`, `fontSize, weightBold` from `../theme/typography`.

```typescript
import React, { useState } from 'react';
import { StyleSheet, Text, View } from 'react-native';
import { useNavigation } from '@react-navigation/native';
import { NativeStackNavigationProp } from '@react-navigation/native-stack';
import { ProteinStackParamList } from '../navigation/TabNavigator';
import { useSafeAreaInsets } from 'react-native-safe-area-context';
import { TabBar } from '../components/TabBar';
import { MacrosView } from '../components/MacrosView';
import { colors } from '../theme/colors';
import { spacing } from '../theme/spacing';
import { fontSize, weightBold } from '../theme/typography';

const TABS = ['Macros', 'Hydration'];

export function ProteinScreen() {
  const insets = useSafeAreaInsets();
  const navigation = useNavigation<NativeStackNavigationProp<ProteinStackParamList>>();
  const [activeTab, setActiveTab] = useState(0); // 0 = Macros (default per TAB-01)

  return (
    <View style={[styles.container, { paddingTop: insets.top }]}>
      <View style={styles.header}>
        <Text style={styles.title}>Nutrition</Text>
      </View>
      <TabBar tabs={TABS} activeIndex={activeTab} onTabPress={setActiveTab} />
      {activeTab === 0 ? (
        <MacrosView navigation={navigation} />
      ) : (
        <View style={styles.placeholder}>
          <Text style={styles.placeholderText}>Hydration coming in Plan 02</Text>
        </View>
      )}
    </View>
  );
}
```
