Note: Since `db` is a `const` export (a Promise), the mock pattern is:
- jest.mock('../database') replaces the module
- Cast `db` as mocked and provide a resolved mock database object
- Cast `executeSql` as jest.MockedFunction and chain .mockResolvedValueOnce() for each SQL call in sequence
