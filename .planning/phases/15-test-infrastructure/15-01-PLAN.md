  <action>
Replace jest.config.js with full configuration. The config must:

1. Keep `preset: 'react-native'` (provides babel-jest transform, RN test environment, etc.)

2. Add `setupFilesAfterFramework: ['./jest.setup.js']` to load global test setup after the RN preset's own setup.

3. Add `coverageReporters: ['lcov', 'text-summary']` so that:
   - lcov writes `coverage/lcov.info` for tooling
   - text-summary prints a human-readable summary to stdout

4. Add `collectCoverageFrom` array:
   ```
   [
     'src/**/*.{ts,tsx}',
     '!src/**/*.d.ts',
     '!src/types/**',
     '!src/theme/**',
     '!src/navigation/**',
   ]
   ```
   This includes all source TypeScript but excludes type-only files, theme constants, and navigation wiring (which are tested via screen tests).

5. Add `coverageThreshold` with global enforcement:
   ```
   {
     global: {
       lines: 80,
       statements: 80,
       functions: 70,
       branches: 70,
     }
   }
   ```
   Lines and statements at 80% per milestone goal. Functions and branches at 70% (more realistic for UI-heavy codebase with many conditional renders).

6. Add `coverageDirectory: 'coverage'` to put output in project root coverage/ folder.

7. Add `moduleFileExtensions: ['ts', 'tsx', 'js', 'jsx', 'json']` (ts/tsx first for faster resolution).

8. Add `testMatch: ['**/__tests__/**/*.(test|spec).(ts|tsx)', '**/*.(test|spec).(ts|tsx)']` to find tests in __tests__/ and co-located.

9. Add `transformIgnorePatterns` override to ensure all RN and community packages are transformed:
   ```
   ['node_modules/(?!((jest-)?react-native|@react-native(-community)?|@notifee|react-native-.*|@react-navigation)/)']
   ```
   This extends the RN preset's default to also include @notifee and @react-navigation.

Create jest.setup.js with:
```javascript
// Silence console.warn and console.error in tests to reduce noise
// Tests that need to verify warnings should mock console explicitly
jest.spyOn(console, 'warn').mockImplementation(() => {});
jest.spyOn(console, 'error').mockImplementation(() => {});

// Silence LogBox (React Native warning overlay)
jest.mock('react-native/Libraries/LogBox/LogBox', () => ({
  __esModule: true,
  default: {
    ignoreLogs: jest.fn(),
    ignoreAllLogs: jest.fn(),
  },
}));
```
