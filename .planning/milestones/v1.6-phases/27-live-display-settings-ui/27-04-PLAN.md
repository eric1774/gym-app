```typescript
const attemptAutoReconnect = useCallback(async () => {

    return;
  }

  const settings = await getHRSettings();
  // ... rest of function unchanged
```

This prevents the initial flicker on workout mount when the HR monitor is already connected from a prior session or Settings screen pairing.

Do NOT change the `disconnect` callback (lines 333-361). The `setCurrentBpm(null)` on line 359 is correct for manual unpair — the user explicitly disconnected, so clearing BPM is the right behavior. Only the automatic disconnect handler (Change 1) should retain the last value.

3. **End Workout stays in Row 1** — it can never be pushed off screen by HR elements since they are in Row 2.
4. **Heart pair button stays in Row 1** — when no device is paired, it still shows in the top row (lightweight, small).
